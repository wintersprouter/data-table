## 使用技術 
  -  HTML排版
  -  CSS variable
  -  CSS  BEM命名
  -  遍歷 DOM 節點
  -  Javascript Event Handler、Event Listener
  
## 實作功能
  -  廣告被選擇時的樣式
  -  固定 table header
  -  一鍵轉換 dark mode
  -  廣告列 hover 效果
  -  一鍵選取所有廣告
